# Forti Sphere

Forti Sphere is a polished, installable security-focused web app experience built as a progressive web app (PWA). It includes a landing page, onboarding, secure login, dashboard, privacy, and terms pages.

## Features
- Security-first landing experience
- Installable PWA
- Offline-ready caching
- Onboarding flow
- Sign-in and dashboard experience
- Privacy and terms pages

## Run locally
Open the project folder and start a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000/

## Deploy to GitHub Pages
1. Push the repository to GitHub.
2. Open the repository Settings.
3. Go to Pages.
4. Choose GitHub Actions as the source.
5. The workflow in [.github/workflows/deploy-pages.yml](.github/workflows/deploy-pages.yml) will publish the site automatically.

## Notes
This is a demo-style product experience and not a real security backend.
